Orchestration Modes Explained
Direct Mode

All requests go directly to the selected model. Simple and predictable. Best when you know which model works for your use case.

Cascade Mode

Tries faster/cheaper models first. Falls back to more capable models if the response is unsatisfactory. Cost-effective for simple queries.

Router Mode

Analyzes each request and routes it to the optimal model based on complexity, task type, and required capabilities. Best for varied workloads.
